# 深度思考创作指令

## 底层逻辑·哲学叙事版

**核心基调**：
不仅仅是技术分析或社会评论，而是寻找现象背后的**底层逻辑**与**哲学隐喻**。

**创作原则**：
1.  **拒绝表象解读**：
    *   不要停留在社会阶层、逆袭、机会主义等世俗层面。
    *   要挖掘更底层的机制（如：随机性、熵增、复杂系统、进化论）。
2.  **跨学科/跨文化联想**：
    *   **数学/科学**：使用数学公理（如 P vs NP）、物理定律（熵、相对论）、生物学机制（突变、自然选择）作为论证基石。
    *   **流行文化**：恰当引用影视/动漫（如《权力的游戏》、《火影忍者》）中的概念作为通俗隐喻，降低理解门槛但提升趣味性。
3.  **宏大叙事与历史感**：
    *   将话题置于生命进化、人工智能发展史、人类文明演进的坐标系中。
    *   强调“不可控性”、“涌现”、“必然性”与“偶然性”的辩证关系。
4.  **具体案例的升华**：
    *   引用具体人物（如李飞飞、图灵）或书籍时，不要做简单的生平介绍，要提炼其思想中与核心论点共鸣的瞬间（如“猝不及防的时代跃迁”、“随机游走式联想”）。

**示例结构**：
*   **引子**：用一个耳熟能详的流行语或现象切入，但立即反转其常规含义。
*   **理论支撑**：引入一个硬核的科学/数学概念（P!=NP）来解释上述反转。
*   **现实映射**：结合当前科技热点（AI、大模型）或历史人物感悟。
*   **升华结语**：回归哲学层面，给出一个开放性或总结性的深刻结论。
